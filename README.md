# Jetpack-Compose-Mobile-App
Proyek eksplorasi mandiri dalam membangun aplikasi Android menggunakan Jetpack Compose. Meskipun memiliki antarmuka (UI) yang sederhana dan minimalis, proyek ini difokuskan pada penerapan arsitektur MVVM, manajemen state yang efisien, serta validasi input real-time untuk memberikan pengalaman pengguna yang solid.
