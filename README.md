# Hikmalmobile 📱

Aplikasi registrasi sederhana yang dibangun menggunakan **Kotlin** dan **Jetpack Compose**. Project ini dibuat untuk mempelajari dasar-dasar UI modern di Android dan manajemen state.

## ✨ Fitur
- **Modern UI:** Menggunakan Material 3 design.
- **Form Validation:** Validasi input email dan password secara real-time.
- **Secure Input:** Fitur sembunyi/tampilkan password (toggle visibility).
- **Architecture:** Menggunakan pola MVVM (Model-View-ViewModel) agar kode lebih rapi.

## 🛠️ Tech Stack
- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Tools:** Android Studio (Ladybug / Koala)

## 📁 Struktur Folder Utama
- `ui.theme`: Pengaturan tema warna dan font.
- `MainActivity.kt`: Entry point utama aplikasi.
- `RegistrationScreen.kt`: Tampilan UI form registrasi.
- `RegistrationViewModel.kt`: Logika di balik layar (validasi & state).

## 🚀 Cara Menjalankan
1. Clone repositori ini.
2. Buka di **Android Studio**.
3. Pastikan SDK Android sudah terinstall.
4. Klik tombol **Run** (Ikon Play hijau) untuk menjalankan di emulator atau device fisik.
